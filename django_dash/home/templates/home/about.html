The Application

Welcome to my prototype sports analytics dashboard

This web application is part of my Msc Computing and IT Management dissertation. Its aim is to show Football statistics in a new way. Rather than presenting the statistics based around your favourite teams and players using traditional raw data format, this application makes use of the most cutting edge visualisation and data analysis libraries to make the user experience more interactive and engaging. This application is a prototype only and currently only has a few visualisations to showcase. Please Register with us to get the full experience.
